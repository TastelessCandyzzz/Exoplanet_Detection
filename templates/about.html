About This Website

This project was developed for the NASA Space Apps Challenge 2025, under the Exoplanet Prediction problem statement.

Our goal is to create a simple, accessible web tool that uses machine learning models trained on real NASA mission data (Kepler/K2/TESS) to predict whether a candidate is likely to be an exoplanet.

Exoplanet discovery helps us understand how planetary systems form, whether Earth-like worlds exist elsewhere, and ultimately whether life might exist beyond our solar system.

Our web application allows users to input planetary parameters and instantly receive a prediction. This bridges the gap between raw scientific datasets and an interactive, user-friendly platform.
